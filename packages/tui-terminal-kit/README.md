# @flowtorio/tui-terminal-kit

Terminal UI abstraction layer for building keyboard-driven TUI applications with Vue reactivity.

## Features

- **LayoutBlock System**: Dynamic block-based layout with independent rendering
- **Input Modes**: Vim-like input modes (Normal, Command, Insert, Select)
- **Reactive State**: Built on Vue's reactivity system
- **Specific Renderers**: Purpose-built renderers for common UI patterns
- **Terminal Abstraction**: Clean abstraction over terminal-kit

## Installation

```bash
pnpm add @flowtorio/tui-terminal-kit
```
